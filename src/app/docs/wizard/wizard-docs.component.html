Wizards guide users through pre-defined steps in a particular order to represent a series of related decisions in a userâ€™s direct workflow. Wizards illustrate the step-wise nature of tasks and reinforce progress toward final goals as users complete steps. Wizards are most useful when users must perform steps in a particular order.

This wizard implementation replaces a previous version that was deprecated because of limitations that interfered with complex configuration tasks. The deprecated wizard used the tabs module within a modal component and set tabStyle="wizard" on the sky-tabset component to guide users through a set of pre-defined steps. We recommend that you create wizards with the progress indicator component instead of this deprectated implementation.
